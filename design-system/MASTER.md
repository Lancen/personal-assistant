# Personal Assistant - 设计系统

**项目**: 注重隐私的个人成长工具（情绪管理、任务管理、知识沉淀）

## 3. 字体系统

**字体选择**: `Inter` + `PingFang SC`（中文）

- **排版体系**: 单字体体系（Inter），通过字重建立层次
- **字重**: 300（轻标题）、400（正文）、500（标签）、600（标题）、700（大标题）
- **行高**: 正文 `1.6`，标题 `1.2-1.4`
- **字体大小**: 正文基础 `16px`，符合可访问性要求

**Google Fonts 引入**:
```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap');
```

## 7. 客户端加密 UI 暗示

## 8. 响应式布局规范

**断点遵循 Tailwind 默认**:
- `< 768px`: 移动端，单列布局
- `768px - 1023px`: 平板，双列布局
- `≥ 1024px`: PC，多栏布局，侧边栏 + 主内容

提供的布局类：

| 类名 | 作用 |
|------|------|
| `.content-container` | 内容容器，最大宽度 1200px，水平居中 |
| `.responsive-grid` | 响应式网格：1 → 2 → 3 列自动适配 |
| `.split-layout` | PC端双栏布局（侧边栏 + 主内容）|
| `.split-layout__sidebar` | 侧边栏，固定宽度 320px，粘性定位 |
| `.split-layout__main` | 主内容区域，自适应宽度 |

## 9. 无障碍（Accessibility）要求

| 要求 | 实现 |
|------|------|
| **键盘导航** | 所有可交互元素支持 Tab 访问 |
| **Focus 状态** | `:focus-visible` 显示 3px 明显外环线 |
| **对比度** | 正文 ≥ 4.5:1，大文字 ≥ 3:1 |
| **减少动效** | 尊重 `prefers-reduced-motion` |
| **图标按钮** | 必须加 `aria-label` |

---

## 10. 动画过渡规范

- 微交互时长：`150-200ms`
- 缓动函数：`cubic-bezier(0.4, 0, 0.2, 1)` (Material 标准)
- 只在颜色/透明度/变换上做过渡，避免触发布局重排
